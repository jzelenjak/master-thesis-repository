# 5G Security Master Thesis Repository

This repository contains the scripts and configuration files for the practical part of the Master thesis "Down to Earth, Up to Space: A Security Architecture Analysis of 5G Terrestrial and Non-Terrestrial Networks" at TU Delft (2024/2025). The thesis document is available in the [TU Delft repository](https://repository.tudelft.nl/record/uuid:052a1c75-60fa-425e-9cae-3aaf25ab2aec).

For the scripts and configuration files related to [UERANSIM](https://github.com/aligungr/UERANSIM), see [ueransim/](./ueransim/) directory.

For the scripts and configuration files related to [OpenAirInterface](https://gitlab.eurecom.fr/oai/openairinterface5g), see [openairinterface/](./openairinterface/) directory.

For the common scripts used for both UERANSIM and OpenAirInterface, see [common-utils/](./common-utils/) directory.

For the instructions, refer to the README files in the corresponding directories.
